# 《当代南方地区宗族关系中的学业竞争》论文项目仓库
Repository for the Academic Paper: *Academic Competition in Clan Relationships in Contemporary Southern China*


## 一、项目介绍
本仓库是《当代南方地区宗族关系中的学业竞争》学术论文的配套管理仓库，用于集中存储论文全文、修订记录、查重报告、实证数据图表及辅助研究资料，实现论文项目的版本追溯、资料统一管理。

本论文聚焦**当代中国南方宗族关系与学业竞争的交织逻辑**，核心研究：南方宗族“崇儒重教”的历史传统如何演变为当代亲属间的学业攀比；宗族亲属网络如何通过资源支持、观念形塑影响学业竞争；城市化、教育内卷背景下宗族关系的现代重构及青年“断亲”现象的关联。


## 二、论文核心结构
论文共8个核心章节，内容框架如下：
1. **绪论**：阐述南方宗族与教育绑定的历史背景、核心概念（宗族/南方宗族/宗族学业竞争）、研究思路与方法；
2. **文献综述**：梳理南方宗族与教育的历史传统、当代亲属网络对学业的影响、南方宗族的区域特殊性等相关研究；
3. **现实表现**：呈现当代南方宗族学业竞争的实际形态（亲属攀比、资源倾斜、宗族奖学等）；
4. **形成机制**：从符号互动论、亲属网络、宗族组织角度，分析宗族学业竞争的生成逻辑；
5. **宗族关系的重构**：探讨城市化、教育内卷下南方宗族关系的弱化与青年“断亲”困境；
6. **结论与思考**：总结宗族学业竞争的当代特征，并提出宗族资源互助功能的转型设想；
7. **致谢**：感谢指导教师、学长姐及相关支持，并结合个人经历阐释研究的现实关联；
8. **参考文献**：涵盖社会学、宗族研究、教育研究领域的中外文献（含11篇核心期刊/学位论文/外文期刊）。
